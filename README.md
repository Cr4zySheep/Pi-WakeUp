## Project Pi-WakeUp

# Setting
'config.json' sample :
'''
{
  "db": {
    "name": "DB-NAME"
  },
  "server": {
    "port": "SERVER-PORT",
    "ip": "SERVER-IP"
  }
}
'''

# To start
Need node.js and then :

'''
git clone https://github.com/Cr4zySheep/Pi-WakeUp.git
cd Pi-WakeUp
node install
node app.js
'''
